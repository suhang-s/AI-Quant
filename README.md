# AI-Quant

量化交易学习与实践项目。

## 内容

### 1. 科大讯飞行情面板
- `index.html` — K 线图与交易量交互式面板（ECharts，离线可看，数据内嵌）
- `kdxf_daily.csv` — 近 2 年日线行情数据（来源：Tushare）

### 2. 双均线策略回测看板
- `backtest_dashboard.html` — 交互式策略回测看板，支持：
  - 股票筛选（6 只股票可选）
  - 均线参数调整（短/长周期）
  - 交易成本、初始资金、仓位、时间窗口可调
  - 16 项指标卡片（年化收益、夏普、索提诺、卡玛、最大回撤、胜率、盈亏比、Alpha/Beta 等）
  - 3 张图表（净值曲线 vs 基准、回撤曲线、K线+均线+买卖信号）
  - 逐笔交易明细表

## 数据说明

- 股票池：科大讯飞(002230.SZ)、中芯国际(688981.SH)、比亚迪(002594.SZ)、长江电力(600900.SH)、南大光电(300346.SZ)、药明康德(603259.SH)
- 区间：2024-07 ~ 2026-07（近 2 年）
- 数据均为前复权日线数据
- 数据来源：[Tushare](https://tushare.pro)
